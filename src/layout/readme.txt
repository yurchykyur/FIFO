layouts/
Содержит повторно используемые компоненты макета. Компонент макета - это компонент, составляющий макет страницы.
 Он часто импортирует такие компоненты, как app-bar, app-footer, app-side-nav.
Если ваш проект, вероятно, будет иметь только один макет, этот каталог может не понадобиться, и компонент макета может находиться в каталоге компонентов.
